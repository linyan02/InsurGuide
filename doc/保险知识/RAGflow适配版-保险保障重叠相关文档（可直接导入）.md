# 文档1：学平险_保障责任与结构.txt

学平险全称学生平安保险，是针对中小学生/幼儿的综合小额保障保险。

核心保障责任：

1. 意外身故/伤残：保额通常3万-10万，按伤残等级比例赔付。

2. 意外医疗：包含门诊+住院，0免赔或低免赔，报销社保内费用，保额一般5000-2万。

3. 住院医疗：疾病住院+意外住院，保额1万-6万，有报销比例限制。

4. 住院津贴：按住院天数补贴，常见50-100元/天。

免赔特点：多数产品0免赔或100元以内低免赔。

门诊责任：普遍包含意外门诊，部分含疾病门诊。

与其他险种关系：学平险≈小额意外险+小额住院医疗险，保额低、覆盖广、价格便宜。


# 文档2：商业补充医疗_通用说明.txt

商业补充医疗多为企业团险/个人小额医疗险，用于补充社保报销后剩余费用。

核心特点：

1. 免赔额：常见0免赔，主打小额医疗报销。

2. 保障范围：以社保内自付部分为主，部分可扩展社保外。

3. 门诊责任：可包含普通门诊（感冒、发烧、小病等），弥补百万医疗险不覆盖小额门诊的短板。

4. 住院责任：覆盖社保报销后的剩余住院费用，与百万医疗形成高低搭配。

5. 保额：门诊保额通常几千元，住院保额1万-5万为主。

适用场景：解决0-1万元小额医疗费用，与百万医疗险、社保形成阶梯报销。


# 文档3：保障重叠_判断逻辑.txt

保障重叠判断核心按「费用区间+险种责任+报销/给付类型」区分：

1. 费用区间划分

- 小额医疗（0-1万元）：以0免赔补充医疗、学平险、门诊险为主。

- 大额医疗（1万元以上）：以百万医疗险为主。

2. 责任重叠判断

- 医疗报销型：同一笔费用不可重复报销，优先社保→补充医疗/学平险→百万医疗。

- 身故/伤残给付型：意外险、学平险、重疾险可叠加赔付，不冲突。

3. 险种优先级

- 小病门诊：补充医疗/门诊险/学平险。

- 小额住院：补充医疗/学平险。

- 大病住院/重疾：百万医疗险+重疾险。

- 身故/意外伤残：意外险/学平险。

4. 重叠风险

- 同时买百万医疗+防癌医疗：癌症医疗责任高度重叠，仅报销一次。

- 学平险+意外险：意外医疗不可重复报销；身故/伤残可叠加赔付。


# 文档4：社保_基础报销说明.txt

社保（城乡居民医保/职工医保）是基础医疗保障，仅覆盖基础住院与门诊，不兜底全部费用。

核心规则：

1. 起付线：住院有报销门槛，未达到起付线部分不报销。

2. 报销比例：按医院等级、社保目录内费用比例报销，等级越高报销比例越低。

3. 封顶线：年度报销有最高限额，超限额部分不报销。

4. 报销范围：仅限社保目录内药品、诊疗、服务项目，自费药/进口药/特需医疗不报。

5. 门诊：普通门诊限额低、报销比例低，大病门诊按住院规则执行。

社保定位：基础打底，大额医疗、自费医疗、高额费用需商业保险补充。


# 文档5：意外险_保障责任与结构.txt

意外险主要保障意外导致的身故、伤残、意外医疗，与疾病无关。

核心责任：

1. 意外身故：一次性赔付保额，成人常见20万-100万。

2. 意外伤残：按1-10级伤残比例赔付，保额同身故。

3. 意外医疗：意外导致的门急诊+住院，0免赔或低免赔，社保内/外报销。

与学平险区别：

- 意外险：侧重身故/伤残高保额，意外医疗责任纯粹。

- 学平险：包含意外+疾病住院，身故/伤残保额低，属于综合小额险。

重叠点：意外医疗不可重复报销；身故/伤残可叠加赔付。


# 文档6：防癌险与防癌医疗_区别与重叠.txt

一、险种区别

1. 防癌险（给付型）：确诊癌症一次性赔保额，与治疗费用无关，属于重疾类。

2. 防癌医疗险（报销型）：癌症治疗费用报销，与百万医疗报销逻辑一致。

二、与百万医疗/重疾险重叠

1. 防癌医疗 vs 百万医疗险

- 百万医疗险普遍包含癌症责任，二者癌症医疗报销高度重叠。

- 同一笔癌症治疗费，只能报销一次，不可重复报。

2. 防癌险 vs 重疾险

- 重疾险已含癌症责任时，再买防癌险属于责任重复，仅增加保额。

- 防癌险可作为重疾险保额补充，不冲突但责任重叠。

三、适用人群

- 防癌医疗：买不了百万医疗的高龄/非标体人群。

- 防癌险：癌症家族史、侧重癌症保障人群。


# 文档7：惠民保_保障范围与百万医疗差异.txt

惠民保是地方普惠型医疗险，价格低、投保宽松，与百万医疗险为互补/替代关系。

核心差异：

1. 免赔额：惠民保免赔额高（常见2万），百万医疗1万。

2. 保障范围：惠民保保障简单，多限社保内+部分特药；百万医疗覆盖住院、手术、特药、质子重离子等，更全面。

3. 报销比例：惠民保报销比例70%-80%；百万医疗100%报销（符合条件）。

4. 投保门槛：惠民保无健康告知/宽松；百万医疗有健康告知。

5. 定位

- 惠民保：兜底、普惠，适合非标体/高龄。

- 百万医疗：全面、足额，适合健康人群，保障力度远高于惠民保。

重叠关系：二者均为报销型，住院费用不可重复报销。


# 文档8：重疾险与防癌险_重叠判断.txt

一、责任重叠判定

1. 重疾险必含「恶性肿瘤——重度」责任，属于核心重疾。

2. 防癌险只保癌症（恶性肿瘤+原位癌等），是重疾险的子集责任。

3. 同时拥有重疾险+防癌险：癌症责任完全重叠，属于重复投保。

二、赔付规则

- 均为给付型，确诊癌症可同时赔付，不冲突。

- 但保障责任高度重合，边际保障价值低。

三、配置建议

1. 已买足额重疾险（含癌症二次赔）：无需再买防癌险。

2. 重疾险保额低/健康告知不通过：可加保防癌险补充癌症保额。

四、与百万医疗关系

- 重疾/防癌险是给付赔钱，百万医疗是报销医药费，三者不冲突、可搭配。
