# InsurGuide 项目说明

## 项目定位

InsurGuide（智保灵犀）是一套**智能保险指南与增强 RAG 系统**，面向保险场景提供：

- 多轮对话与知识库问答（RAGflow + 自研增强流水线）
- 意图识别、问题改写、召回、融合、精排等可配置链路
- 合规校验与审计日志
- PC Web 端使用，API 可复用于小程序等终端

## 技术栈

| 层次 | 技术选型 |
|------|----------|
| 后端 | Python 3.9+、FastAPI |
| 配置 | pydantic-settings、.env |
| 数据 | MySQL（业务+日志）、Redis（对话上下文） |
| 检索 | RAGflow（主知识库）、ChromaDB（规则/示例向量）、可选 Elasticsearch |
| 增强 RAG | 自研流水线：意图 → 改写 → 召回 → 融合 → 精排 → 生成与合规 |
| 前端 | PC Web（静态页 + JS 调用 API，可替换为 Vue/React 工程） |

## 快速开始

1. 配置：复制 `.env.example` 为 `.env` 并填写（数据库、Redis、RAGflow、API Key 等）。
2. 依赖：`pip install -r requirements.txt`。
3. 启动 API：`python main.py` 或 `uvicorn api.main:app --host 0.0.0.0 --port 8000`。
4. 访问：API 文档 http://localhost:8000/docs ；PC Web 见 `web/` 目录说明。

## 与小程序扩展

当前 API 设计为无状态、RESTful，认证采用 JWT。小程序端复用同一套接口即可，需在服务端配置 CORS 与小程序合法域名，并在小程序侧维护 token 与 `user_id`（或 openid 映射）用于多轮对话与鉴权。
